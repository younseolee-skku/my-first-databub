## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   cd your-repo-name
   ```

2. **Install dependencies**
   ```bash
   pip install streamlit
   ```

3. **Run the app**
   ```bash
   streamlit run app.py
   ```

4. Open your browser and go to `http://localhost:8501`
